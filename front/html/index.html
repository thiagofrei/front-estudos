Formatação de Texto

aqui vão algumas tags de formatação de texto. a tag b deixa em negrito. porém a tag nao tem importância para o google. a tag strong também deixa em negrito, mas tem importância para o google, tem significado. a tag i deixa em itálico, mas o google prefere a tag em, tem semântica. a tag mark marca algo. a tag small diminui o texto. a tag del faz um risco. a tag ins sublinha o texto. a tag sub coloca a palavra um pouco abaixo do alinhamento, que pode ser usado para citação. a tag sup coloca a palavra um puco acima do alinhamento, pode ser usado para exponenciação.
